== Utilisation de openmp (parallélisation)

Pour compiler avec openmp inclus il faut rajouter `-fopenmp` dans les options de compilation

Comparaison time matmul :
* Sans openmp : real	0m3.304s
* Avec openmp : real	0m1.574s
